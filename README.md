# 学校介绍

这是一个用于展示学校基本信息的微信小程序项目，旨在为学生、家长及访客提供直观、便捷的学校概况查询入口。

## 项目功能

- **学校概况展示**：核心的学校简介信息呈现
- **模块化导航**：包含图片、页面、功能按钮等基础展示模块
- **移动端适配**：专为微信小程序环境设计，体验流畅

## 开发环境

- **开发工具**：微信开发者工具（WeChat DevTools）
- **开发框架**：微信小程序原生框架
- **运行环境**：微信客户端

## 项目结构
├── components/ # 自定义组件（如导航栏）
├── images/ # 图片资源文件夹
├── pages/ # 小程序页面文件夹
├── app.js # 小程序逻辑入口
├── app.json # 全局配置文件
└── app.wxss # 全局样式文件

## 使用方法

1. 打开【微信开发者工具】
2. 选择【导入项目】
3. 填写你的小程序 AppID（测试可使用“测试号”）
4. 选择本项目文件夹，点击编译即可预览
